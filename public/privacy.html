Overview

This Privacy Policy describes how Lincoln Labs, Inc. ("Lincoln Labs," "we," or "us") handles information within the Lincoln Labs Revenue Dashboard (the "Tool"). The Tool is an internal business application restricted to authorized Lincoln Labs personnel. It is not a public-facing product.

By using the Tool, you acknowledge that you have read and understood this policy.

Information We Access

The Tool connects to QuickBooks Online (QBO) via OAuth 2.0 on your behalf. When you authenticate, it accesses the following data from your QBO account:

  • Profit & Loss report data, including income and expense account balances
  • Company realm ID used to scope API requests to the correct QBO company
  • OAuth access and refresh tokens required to maintain your authenticated session

No data is collected from end users beyond what is necessary to authenticate with QBO and render the dashboard. The Tool does not collect names, email addresses, IP addresses, or device identifiers.

How Information Is Used

Data accessed through the QBO API is used exclusively to:

  • Render revenue charts and period-over-period comparisons within the dashboard
  • Maintain an authenticated session so you do not need to re-authorize on every visit

Financial data is not stored persistently. It is fetched live from QBO on each page load and held only in server memory for the duration of the request. It is never written to a database, logged to disk, or transmitted to any third party.

Session Data

Your OAuth tokens and QBO realm ID are stored in a server-side session (backed by an encrypted SQLite store on the host server). Session cookies are:

  • Transmitted over HTTPS only
  • Set with SameSite=Lax to limit cross-site exposure
  • Valid for 7 days, after which you will need to re-authenticate

You can end your session at any time by clicking "Disconnect QBO" in the dashboard header, which destroys the server-side session and clears your tokens.

Data Sharing

Lincoln Labs does not sell, rent, or share your financial data or session credentials with any third party. The only external service the Tool communicates with is the Intuit QuickBooks Online API, solely to retrieve data you have authorized.

Access Controls

The Tool is deployed on a private Railway instance and is intended exclusively for authorized Lincoln Labs team members. Access is controlled by QuickBooks Online OAuth — only accounts with existing access to the Lincoln Labs QBO organization can authenticate successfully.

Data Retention

OAuth tokens stored in your session are automatically expired after 7 days. No financial data is retained beyond the life of a single API request. If you revoke the Tool's access within QuickBooks Online, all stored tokens become immediately invalid.

Changes to This Policy

We may update this policy as the Tool evolves. The effective date at the top of this page reflects the most recent revision. Continued use of the Tool after changes constitutes acceptance of the updated policy.

Contact

Questions about this policy can be directed to the Lincoln Labs team internally. There is no public support channel for this tool.
